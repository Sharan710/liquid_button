## Getting Started
* Add this to your pubspec.yaml
  ```
  dependencies:
  liquid_button: 1.0.0

  ```
* Get the package from Pub:

  ```
  flutter packages get
  ```
* Import it in your file

  ```
  import 'package:liquid_button/liquid_button.dart';
  ```
